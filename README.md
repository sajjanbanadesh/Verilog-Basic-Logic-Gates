🔌 Basic Logic Gates Using Verilog
📌 Project Description
This project demonstrates the implementation of basic logic gates using Verilog HDL.

The gates implemented are:
- AND Gate
- OR Gate
- NOT Gate
- XOR Gate

---
⚙️ Features
- Simulates logic gate behavior
- Verifies output using testbench
- Displays waveform for analysis

---
🧠 Concepts Used
- Verilog HDL
- Gate-level modeling
- Combinational logic
- Simulation and waveform analysis
